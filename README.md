# Ejectuar en desarrollo

1. Clonar el repositorio
2. Ejectuar
```
yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos (mongo 4.4.6 por ser base x86)

```
docker-compose up -d
```

## Stack usado
* MongoDB
* Nest
